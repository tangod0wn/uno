
<h1> Setup </h1>

``` npm run dev ``` 
<br>
local server will then be running on localhost:8000

<h1> Features </h1>
<br>
<h3>
* Animated form for handling user input.
* On submission, JS is used to send that data to the API and retrieve the results.
* The results are then rendered on the page instead of the form.</h3>
